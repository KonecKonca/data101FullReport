Ambari:
	Один из основных элементов Ambari - dashboard, который содержит вкладки:	
- метрики(разлиного рода обобщённая информация(сводка данных по кластеру) с разнымим 
	графиками и т.д), 
- информацией о кластере(какие узлы кластера в рабочем состояние, какие нет и т.д),
- вкладка с историей изменений конфигурации.
	Также в Ambari есть вкладочка с информацией о сервисах, с указание поднят сервис, или нет,
соответсвенно есть навигации по всем перечисленным сервисам с более подробной информацией.
	В правом верхнем углу есть кнопочка для авторизации, через неё можем менять пользователей,
соответсвенно этим и меняя возможные операции.
	
cloudera manager:
	В левой части экрана распологается список сервисов, по которому пожно перемещаться
для получения более детальной информации о каждом сервисе.
	Правее находится вкладка All health issues(информация о узлах кластера)
	Следующая вкладка configuration - где находится конфигурационная информациия с возможностью
изменения конфигурации.
	Сверху по центру есть вкладка charts с различными обобщенными графиками, 
отражающие состояние кластера(аналогична метрика из Ambari)
